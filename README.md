# **Track!t - Sistema de Gestión de Inventarios**

**Sistema web completo para gestión de inventarios y ventas diseñado para pequeñas empresas.**

---

## **Descripción**

Track!t es una aplicación web que permite:

- Registro y seguimiento de productos en inventario  
- Procesamiento de ventas con actualización automática de stock  
- Generación de reportes de ventas e inventario  
- Alertas automáticas para productos con bajo stock  
- Gestión de usuarios con roles (**Admin/Vendedor**)  
- Gestión de categorías de productos  

---

## **Backend**

- **Java (JDK 11 o superior)**  
- **Jakarta Servlet API**  
- **JDBC** para conexión a base de datos  
- **Patrón DAO (Data Access Object)**  
- **Patrón MVC (Model-View-Controller)**  
- **Apache Tomcat**

---

## **Frontend**

- **JSP (JavaServer Pages)**  
- **HTML, CSS**  
- **JavaScript**  
- **Bootstrap 5.1.3**  
- **Font Awesome 6.0**

---
